# tw-simple-bmi-calculator

A simple BMI calculator built using HTML5, CSS3 with Tailwind CSS and JavaScript.

# To Use 

Clone or Download the repository
`git clone https://github.com/MAbdurahman/tw-simple-bmi-calculator.git`

Install the dependencies

`npm install or yarn`

To watch the CSS files
(watches the `src/input.css` file for changes)

`npm run watch or yarn watch`

To build the project, run the following command
(builds to `css/main.css`, which is the final css file)

`npm run build or yarn build`
